This section covers everything you need to implement new Connectors, Stages, and Indexers for Lucille.

You can develop new components for use in your own Java project that includes Lucille as a dependency — the Lucille framework will discover and run them for you without any changes to the core codebase. You can also develop components with the intent of contributing them back to the Lucille project itself. This guide is useful in both cases. If you are contributing back to Lucille, read the [Contributor Guide]({{< relref "docs/contributing" >}}) as well.
