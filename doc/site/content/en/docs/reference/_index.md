The Reference section is a catalogue of everything Lucille ships with. For conceptual explanations of how these components work and how they fit together, see [Architecture]({{< relref "docs/architecture" >}}).

- [Connectors]({{< relref "docs/reference/connectors" >}}) — built-in connectors for files, databases, Kafka, RSS, and Solr, plus File Handlers.
- [Stages]({{< relref "docs/reference/stages" >}}) — complete stage catalogue organized by category, plus detailed pages for complex stages.
- [Indexers]({{< relref "docs/reference/indexers" >}}) — Solr, OpenSearch, Elasticsearch, CSV, and plugin indexer configuration.
- [Plugins]({{< relref "docs/reference/plugins" >}}) — optional extension modules with heavyweight dependencies.
- [Glossary]({{< relref "docs/reference/glossary" >}}) — definitions of key terms used throughout the documentation.
- [JavaDocs]({{< relref "docs/reference/javadocs" >}}) — generated API documentation for all public classes and methods.
